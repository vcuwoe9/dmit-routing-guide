## CMIN2 是什么，DMIT 移动回程为什么值得关注

先解释一下术语。CMIN2（AS58807）是移动在精品网方向专门建的国际线路，和电信的 CN2 GIA 定位对等，都属于付费精品路由。普通移动出口走的是 CMI（AS58453），质量差几个档次——晚高峰时期抖动大，延迟不稳定。

DMIT 的 Eyeball 系列是目前市面上少数把 CMIN2 用作移动回程的商家之一。实测路由走法是：VPS 流量从洛杉矶出来后，直接打到 AS58807 的 CMIN2 节点（223.120.197.x 这段），从上海入境，全程不绕第三方 Tier 1，延迟压得住。

移动用户用下来，洛杉矶到国内延迟通常在 160-170ms 左右，晚高峰波动不大。对比那些挂着"CN2"标签但移动实际走 163 骨干的机器，差距明显。

**Tier 1（LAX.T1）**——国际互联线路，无中国大陆专项优化。电信双程 CT163，联通 CU169（AS4837），移动双程 CMI。价格最便宜，适合面向海外用户的业务，或者流量需求大但对回国速度没要求的场景。

简单说：
- 电信用户追求极致 → Premium
- 移动/联通用户要性价比 → Eyeball
- 流量大但不在意回国速度 → Tier 1

## 移动回程实测：延迟和路由跑出来是什么样

拿 CMIN2 路线来说，路由跳数大概是这样的走法：

1. 从 DMIT 洛杉矶骨干（193.41.250.x）出发
2. 打到 AS58807 的 CMIN2 节点（223.120.197.x，洛杉矶落地）
3. 经上海中转（223.120.161.x）入境国内
4. 进入移动 AS9808 骨干到目的地

整条路径下来，广州方向延迟大概在 155-165ms，上海方向在 125-130ms。这个数据在用移动宽带的场景下，算是相当稳了——没有绕日本 NTT 或者走美国东岸那种 250ms 起步的情况。

电信用户用 Eyeball 走的是 AS9929/CMIN2 负载均衡，不如走纯 CN2 GIA 的 Premium 干净，但晚高峰稳定性还是比 163 好不少。联通用户走 AS9929 精品网，体验也不差。

说实话，如果你主力宽带是移动，Eyeball 比 Premium 更划算。同样是 $9.99/月，EB.TINY 给你 2Gbps 带宽和 1.5T 流量，Pro.TINY 才 1Gbps 和 1T 流量，而且移动回程 Premium 其实走的是 GIA，不是原生移动精品路由，CMIN2 对移动用户来说线路属性更对口。

## 常见问题解答

**Q：DMIT 移动回程走 CMIN2 还是 CMI，有什么区别？**

CMIN2（AS58807）是移动的精品网线路，相当于移动版的 CN2 GIA，带宽有独立保障，延迟更稳。CMI（AS58453）是普通移动国际出口，质量差很多，晚高峰更容易抖。Eyeball 系列走 CMIN2，不是普通 CMI。

**Q：我是电信用户，买 Eyeball 还是 Premium？**

Premium 对电信的体验更纯粹——去程 CN2 GIA，回程也是 CN2 GIA，双向优化。Eyeball 电信回程走 AS9929/CMIN2 负载均衡，不如 GIA 干净，价格便宜但电信体验略差。有预算就选 Premium，预算紧选 Eyeball 也能用。

**Q：套餐流量超出后怎么办？会停机吗？**

不停机。超出月流量限额后自动切到低速限速模式（通常 4-10Mbps 不限量），服务持续运行。下个计费周期开始后恢复正常速度。

**Q：DMIT 支持哪些付款方式？**

支付宝、微信支付、PayPal 都支持，国内用户付款没有障碍。

**Q：EB.TINY 和 Pro.TINY 同价 $9.99，该选哪个？**

移动/联通用户选 EB.TINY（CMIN2+9929 回程，带宽 2Gbps，流量 1.5T）。电信用户更在意线路纯净度可以选 Pro.TINY（CN2 GIA 回程，带宽 1Gbps，流量 1T）。单纯看配置 EB.TINY 更划算，电信用户看线路就选 Pro。
